# minimal-rl
RL algorithms with minimal lines of code

## Installation
```console
$ git clone https://github.com/r-salas/minimal-rl.git
$ cd minimal-rl
$ pip install -r requirements.txt
```

## DQN
```console
$ python dqn.py
```
